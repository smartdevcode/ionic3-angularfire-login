# Integrating Firebase with AngularFire2 into Angular2 & Ionic2
---

###Update npm modules
```
npm install 
```

--

###Use your own Firebase Account
You will need to change the following code in `app.ts`
```
defaultFirebase('https://[YOUR-APP-NAME].firebaseio.com/'),
```
